Ⅱ  签证官业务审查流程不规范      
        签证官递出的资料显示，因为214B条款拒绝投诉人签证。    投诉人在签证准备中，按照大使馆网站签证页面--如何申请--准备签证申请所需材料中所提到的材料进行准备。 第六点提到需要准备：“支持性文件，象英文简历等，请点击这里看范例。同时请点击这里查看不同签证种类的具体说明。 ”  

       投诉人按照锚文本导航（请点击这里查看不同签证种类的具体说明）-签证种类-学生签证（F和M签证）   中提到的十三个款项进行准备，并将每一款项准备的证明材料列成清单。为方面签证官审查，将清单打印在A4纸上，准备递交给签证官。

      大使馆网页“怎样决定是否颁发一个签证”提到： 大多数的拒签都是因为申请人无法证明自己在境外有不打算放弃的住所。申请人必须证明他们具有迫使其在美短期停留后必须回国的约束力来证明这个住所的存在。 依据美国法律， 证明这一点是每一位申请人的责任.”
